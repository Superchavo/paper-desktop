Paper Desktop v1.2

A lightweight desktop environment for Fedora (aarch64/noarch) developed by Superchavo.

🚀 Installation via Repository

1. Add the repository to your system by creating the config file:

sudo nano /etc/yum.repos.d/paper-desktop.repo

2. Paste the following content into the file:

[paper-desktop]
name=Paper Desktop Repository
baseurl=https://superchavo.github.io/paper-desktop/repo/
enabled=1
gpgcheck=0

3. Now you can install it directly using DNF!

sudo dnf clean all
sudo dnf install paper-desktop
