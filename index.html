📄 Paper ENVIRONMENT

A professional Windows 10-style desktop experience optimized for Fedora Proot.

Welcome to the official repository for Paper ENVIRONMENT. This project delivers a lightweight and productive graphical interface, specifically designed to run within Proot containers on ARM and x86 hardware.

🚀 Installation & Setup Guide

Follow these steps in your Fedora terminal to install the environment correctly:

1Add the Repository:

su -c 'curl -sL https://raw.githubusercontent.com/Superchavo/paper-desktop/main/paper-environment.repo -o /etc/yum.repos.d/paper-environment.repo'

2Install the Package:

su -c 'dnf clean all && dnf install paper-desktop -y'

3Initialize User Configuration:

You must copy the base configuration to your home directory:

cp /etc/skel/.jwmrc ~/.jwmrc

4Launch the Environment:

Set your VNC display variable and start the session manager:

export DISPLAY=:1
paper-session
